# 爱优企服（AIYOU）网站项目分析报告

## 1. 项目概述

这是一个名为**爱优企服（AIYOU Business Group）**的企业网站项目，主要服务于中俄双边贸易，帮助中国企业拓展俄罗斯市场。网站提供了企业介绍、业务领域、展会计划、联系我们等功能模块，并集成了AI智能咨询功能。

## 2. 技术栈介绍

### 前端技术栈
- **React 19**: 用于构建用户界面的JavaScript库

- **Google Gemini API**: 用于AI智能咨询功能的语言模型API

### 开发工具

- **npm**: 包管理工具

## 4. React基础概念解释

### 什么是React？
React是一个用于构建用户界面的JavaScript库，它允许开发者创建可复用的UI组件。

### 核心概念

1. **组件**: React应用由组件组成，每个组件都是独立的、可复用的UI单元。
   - 例如：`AIConsultant.tsx`是一个AI咨询聊天组件

2. **JSX**: 一种JavaScript的语法扩展，允许在JavaScript中编写类似HTML的代码。
   ```jsx
   <div className="container">
     <h1>Hello World</h1>
   </div>
   ```

3. **状态(State)**: 组件内部的数据，用于存储组件的动态信息。
   - 例如：`const [isOpen, setIsOpen] = useState(false);` 用于控制聊天窗口的显示/隐藏

4. **属性(Props)**: 从父组件传递给子组件的数据。
   - 例如：`props.title`可以将标题传递给子组件

5. **钩子(Hooks)**: React提供的函数，用于在函数组件中使用状态和生命周期功能。
   - `useState`: 用于管理组件状态
   - `useEffect`: 用于处理副作用（如API调用、DOM操作等）
   - `useRef`: 用于访问DOM元素

## 5. 核心功能模块分析

### 5.1 主应用组件 (App.tsx)

`App.tsx`是整个应用的主组件，包含了网站的所有主要内容和功能：

- **导航栏**: 包含公司Logo、导航链接和语言切换功能
- **英雄区**: 展示公司核心价值主张
- **服务展示**: 介绍公司的核心业务
- **行业聚焦**: 展示公司专注的行业领域
- **展会计划**: 仅在中文版本显示，展示公司的展会安排
- **关于我们与联系**: 提供公司介绍和联系方式

### 5.2 AI智能咨询 (AIConsultant.tsx)

AI咨询组件是一个交互式聊天功能，允许用户与AI顾问对话：

- **聊天窗口**: 显示用户和AI的对话历史
- **消息输入**: 用户可以输入问题
- **AI响应**: 通过Google Gemini API获取智能响应
- **实时反馈**: 显示加载状态和成功/错误提示

### 5.3 国际化支持 (locales.ts)

网站支持三种语言：中文、英文和俄文：

- **翻译文件**: `locales.ts`包含所有语言的翻译内容
- **语言切换**: 用户可以通过导航栏的下拉菜单切换语言
- **条件渲染**: 根据当前语言显示不同内容（如仅中文显示展会计划）

### 5.4 路由管理

虽然当前项目中路由功能实现较为简单，但已经引入了`react-router-dom`用于页面导航：

- **页面组件**: 每个页面都有对应的组件（如Home.tsx、ServicesPage.tsx等）
- **导航链接**: 使用`<Link>`组件实现页面间的跳转

## 6. 核心代码分析

### 6.1 React组件的基本结构

```tsx
// 导入必要的库和组件
import React, { useState } from 'react';

// 定义组件
const ComponentName: React.FC = () => {
  // 定义状态
  const [count, setCount] = useState(0);

  // 定义事件处理函数
  const handleClick = () => {
    setCount(count + 1);
  };

  // 返回JSX（组件的UI）
  return (
    <div>
      <h1>Count: {count}</h1>
      <button onClick={handleClick}>Increment</button>
    </div>
  );
};

// 导出组件
export default ComponentName;
```

### 6.2 AI咨询组件的核心逻辑

```tsx
// 处理发送消息
const handleSend = async () => {
  if (!input.trim() || isLoading) return;

  // 添加用户消息到聊天记录
  const userMsg: ChatMessage = { role: 'user', text: input, timestamp: new Date() };
  setMessages(prev => [...prev, userMsg]);
  setInput('');
  setIsLoading(true);

  try {
    // 调用Gemini API获取AI响应
    const response = await generateConsultationResponse(input, "The user is visiting AIYOU's website.");
    // 添加AI响应到聊天记录
    const modelMsg: ChatMessage = { role: 'model', text: response, timestamp: new Date() };
    setMessages(prev => [...prev, modelMsg]);
  } catch (err) {
    // 处理错误情况
    const errorMsg: ChatMessage = { role: 'model', text: "Sorry, I am busy right now.", timestamp: new Date() };
    setMessages(prev => [...prev, errorMsg]);
  } finally {
    setIsLoading(false);
  }
};
```

## 7. 项目运行与开发

### 运行项目

1. 安装依赖：
```bash
npm install
```

2. 启动开发服务器：
```bash
npm run dev
```

3. 在浏览器中访问：`http://localhost:5173`

### 构建项目

```bash
npm run build
```

构建后的文件将生成在`out`目录中，用于部署到生产环境。
